argument-hint: "[idea-name] (省略時は全アイデア一覧)"

アイデアの進捗状況を表示する。

## 手順

### 引数なしの場合（全アイデア一覧）

1. `ideas/` 配下の全ディレクトリを列挙する
2. 各アイデアの `README.md` からステータステーブルを読み取る
3. 以下の形式で一覧を表示する:

```
| アイデア | 現在のフェーズ | 最終更新 |
|---------|--------------|---------|
| <name> | <phase> | <date> |
```

4. アイデアが存在しない場合は `/new-idea` の使い方を案内する

### 引数ありの場合（特定アイデア）

1. `ideas/$ARGUMENTS/README.md` を読む
2. 全フェーズのステータスを表示する
3. 現在進行中のフェーズについて:
   - そのフェーズのドキュメントの作成状況を確認する
   - 次にやるべきことを提案する
4. 存在しないアイデア名が指定された場合は、利用可能なアイデア一覧を表示する
